### Iteration & Loops
#### for...of
```
for ([initialExpression]; [conditionExpression]; [incrementExpression]) {
  statement  
}
```
#### while
```
while (condition) {
  statement
}  
```
#### do...while
```
do {
  statement
} while (condition);
```
#### for...in
```
for (variable in object) {
  statement
}
```
#### for...of
```
for (variable of object) {
  statement
}
```
